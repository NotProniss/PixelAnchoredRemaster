# ComfyUI Pixel Anchored Remaster

A compact ComfyUI custom node that wraps the Pixel Anchored Remaster workflow into a single node.

## Features

- IMAGE in, IMAGE out
- Tiled VAE encode/decode
- Remaster-only path
- Hidden pad color, fixed to white
- Visible sampler / scheduler / upscale controls

## Install

Drop this folder into `ComfyUI/custom_nodes/` and restart ComfyUI.

## Node

- `Pixel Anchored Remaster`

## Notes

This node is meant to mirror the workflow branch closely while staying easy to drop into ComfyUI as a reusable custom node.
